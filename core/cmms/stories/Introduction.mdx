# ChatterFix CMMS Design System

Welcome to the ChatterFix CMMS component library! This Storybook contains all the UI components, patterns, and guidelines for building consistent and powerful maintenance management interfaces.

## üéØ Overview

ChatterFix CMMS is an AI-powered Computerized Maintenance Management System that helps facilities manage work orders, assets, inventory, and maintenance schedules. This design system ensures all components follow consistent styling and behavior patterns.

## üé® Design Principles

### **Dark Theme First**
- Primary background: `#0a0a0a` to `#16213e` gradient
- Purple accent colors: `#667eea` to `#764ba2`
- Glass morphism effects with backdrop blur
- High contrast for accessibility

### **Maintenance-Focused UX**
- Clear status indicators with color-coded badges
- Priority-based visual hierarchy
- Quick action buttons for common tasks
- Responsive design for mobile technicians

### **AI-Enhanced Interface**
- Smart defaults and suggestions
- Predictive text and autocomplete
- Contextual help and insights
- Automated workflow indicators

## üìö Component Categories

### **Core Components**
- **Buttons**: Primary, secondary, and danger actions
- **Cards**: Content containers with hover effects
- **Forms**: Input fields, selects, and form layouts
- **Navigation**: Navbar, breadcrumbs, tabs, and side navigation

### **CMMS-Specific Components**
- **Work Order Management**: Lists, cards, and forms
- **Asset Management**: Grid views and detail cards
- **Parts Inventory**: Stock levels and reorder alerts
- **Maintenance Scheduling**: Calendar and list views
- **KPI Dashboards**: Metrics and performance indicators

## üîß Usage Guidelines

### **Component Integration**
```javascript

// Use ChatterFix CMMS styling classes
<div className="chatterfix-form">
  <Button variant="primary">Create Work Order</Button>
```

### **Styling Classes**
- `.btn-primary`, `.btn-secondary`, `.btn-danger` - Button variants
- `.card`, `.feature-card`, `.stat-card` - Card types
- `.form-group`, `.form-input`, `.form-label` - Form elements
- `.status-badge` with modifiers - Status indicators
- `.chatterfix-table` - Data tables

### **Color Palette**
- **Primary**: `#667eea` (ChatterFix Purple)
- **Secondary**: `#764ba2` (Deep Purple)
- **Success**: `#10b981` (Green)
- **Warning**: `#f59e0b` (Amber)
- **Error**: `#ef4444` (Red)
- **Info**: `#3b82f6` (Blue)
