# ChatterFix Infrastructure

Kubernetes, Docker, and deployment infrastructure for ChatterFix CMMS.

## Structure
- `k8s/` - Kubernetes manifests and configurations
- `docker/` - Docker configurations and compose files
- `config/` - Service configurations and environment files
- `scripts/` - Infrastructure and deployment scripts
  - `deploy/` - Production deployment scripts
  - `dev/` - Development tools and scripts
  - `vm/` - VM startup and configuration scripts
  - `testing/` - Test execution scripts
  - `platform/` - Platform management scripts
- `monitoring/` - Monitoring and observability tools
- `dev/` - Development infrastructure tools

## Deployment
See `scripts/deploy/` for deployment options.
